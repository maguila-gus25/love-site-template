# Screenshots

Drop your website screenshots in this folder using these exact filenames so they
appear automatically in the main [README](../../README.md) Preview section.

| Filename | What to capture |
|---|---|
| `01-hero.png` | Hero section (names + falling petals) |
| `02-timeline.png` | "How it all started" timeline |
| `03-gallery.png` | "Our moments" polaroid gallery grid |
| `04-lightbox.png` | A photo opened in the gallery lightbox |
| `05-reasons.png` | "Reasons I love you" card grid |
| `06-counter.png` | "Days together" counter |
| `07-vinyl.png` | Vinyl turntable / Spotify player |
| `08-letter.png` | The opened animated letter |
| `09-date-voucher.png` | The spin-the-wheel screen |
| `10-voucher-ticket.png` | The final date voucher ticket |

Tips:
- PNG or JPG both work; ~1000px wide looks good on GitHub.
- If you prefer different names, just update the links in the main README too.

After adding the files:

```bash
git add docs/screenshots
git commit -m "docs: add screenshots"
git push
```
